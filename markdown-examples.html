一、请假申请 ​

1、请假申请单查询 ​

  • 接口描述

    根据工号查询当前登录人的请假申请单

  • 请求路径

    S_KQ_16

  • 请求方法

    POST

  • 数据提交方式

    application/json

  • 请求参数

    参数名称参数描述数据类型备注
    staffId当前登录人工号String必填
    pageNo页号Number非必填（默认 1）
    pageSize页大小Number非必填（默认 10）

  • 请求参数举例

    json
    {
  "staffId": "122991",
  "pageNo": 1,
  "pageSize": 10
}

  • 返回参数

    参数名称参数描述数据类型备注
    name姓名String
    offworkDays请假时长String
    bDate请假开始日期Stringyyyy-MM-dd
    eDate请假结束日期Stringyyyy-MM-dd
    offworkKindName请假类型String
    statusName申请单状态String录入中/审批中/审批完成/作废
    guid申请单主键String

  • 返回参数举例

    json
    {
  "code": 0,
  "msg": "查询成功!",
  "data": {
    "result": [
      {
        "name": "许智平",
        "offworkDays": "1",
        "bDate": "2023-11-29",
        "eDate": "2023-11-29",
        "offworkKindName": "事假",
        "statusName": "录入中",
        "guid": "f3db1263-399f-445a-89ba-22bf49db9a54"
      }
    ]
  }
}
